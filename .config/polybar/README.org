** Modules
*** Uptime (session-time)
This module uses a custom script to display in text system uptime in human readable format.
#+begin_src conf
[module/session-time]
type = custom/script
exec = ~/dotfiles/.config/polybar/session.sh
interval = 1
#+end_src
